## ⏱️ Timing Breakdown

```
0ms     → User clicks/navigates
0-10ms  → Component mounts
10ms    → Skeleton appears (instant)
10-500ms → Data fetching (varies by connection)
500ms   → Data received
500-600ms → Smooth transition to content
600ms   → User can interact
```
