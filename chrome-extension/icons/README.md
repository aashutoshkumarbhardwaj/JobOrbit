# Extension Icons

This directory contains the icon files for the Job Orbit Chrome Extension.

## Required Icons

The following icon sizes are required by Chrome Extension manifest v3:

- `icon-16.png` - 16x16 pixels (toolbar icon)
- `icon-32.png` - 32x32 pixels (Windows computers)  
- `icon-48.png` - 48x48 pixels (extension management page)
- `icon-128.png` - 128x128 pixels (Chrome Web Store and installation)

## Creating Icons

1. Start with a high-resolution SVG or PNG of the Job Orbit logo
2. Ensure the design is readable at small sizes (16px)
3. Use a transparent background
4. Export each size with appropriate optimization
5. Test the icons in the extension popup and Chrome toolbar
