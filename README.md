## Why JobOrbit

Most job trackers are glorified spreadsheets with a prettier UI. You still have to manually log every application, update every status, remember every deadline.

JobOrbit is different because it's built as part of the **AI Job Application Assistant** ecosystem. When the extension fills your application, JobOrbit is already logging it. When you get a reply, you mark it in one click. When you have an interview, the full context — job description, your resume version, the cover letter you used — is right there.

```
Without JobOrbit          →     With JobOrbit
──────────────────────────────────────────────────────────────
"Did I apply here?"       →     Yes. 12 days ago. Here's what you wrote.
"What stage is this?"     →     Final round. Interview is Tuesday at 2pm.
"Which resume did I use?" →     Version 3 — the DevOps-focused one.
"Did they ghost me?"      →     Last contact: 8 days ago. Follow up?
```

## What It Does

### 📍 One-Click Application Logging
When you use the AI Job Application Assistant to fill a form, JobOrbit logs it automatically. Applied manually? Add it in two taps. No copy-paste, no spreadsheet columns to fill.

### 🗂️ Full Application Context
Every application stores the complete picture — job description snapshot, company details, resume version used, cover letter, notes, and every interaction on a timeline.

### 📊 Status Pipeline
Move applications through a visual pipeline that actually matches how hiring works:

```
Applied  →  Screening  →  Interview  →  Offer  →  Accepted
                                ↓
                         Rejected / Ghosted / Withdrawn
```

No status gets lost. No application disappears into the void.

### ⏰ Smart Follow-Up Reminders
JobOrbit watches for silence. If a company hasn't responded in 7 days, you get a nudge — not a spam notification, but a contextual reminder with the original job post and your application attached.

### 📈 Your Search, At a Glance
- How many applications this week vs last week
- Your response rate by job type, company size, or location
- Average time from application to first response
- Which resume version gets the most callbacks

### 🔗 Works With or Without the Extension
JobOrbit is a standalone product. You don't need the autofill extension to use it. But when you use both, the workflow is seamless — apply in seconds, track automatically, never lose context.

## Roadmap

This is the public roadmap. 50+ people on the waitlist helped shape it.

**v1 — Launch**
- [ ] Application dashboard with pipeline view
- [ ] Manual and auto-logging from the extension
- [ ] Job description snapshots (so you can read it again even if the listing goes down)
- [ ] Status tracking with timeline per application
- [ ] Follow-up reminders

**v2 — Intelligence**
